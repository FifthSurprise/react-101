Lifecycle Events

Components have a lifecycle as they are mounted, updated, and unmounted from the DOM. React components allow you to tap into the lifecycle with "lifecycle methods". These methods are useful for things like starting timers, fetching data from a server, or using third-party libraries like jQuery plugins or data visualization libraries.

You should already be familiar with 2 lifecycle methods: render() and constructor(). Here are a few more you should be aware of:

  • componentWillMount()
  • componentDidMount()
  • shouldComponentUpdate()
  • componentWillUnmount()
