Using Third-Party Libraries

While React abstracts the DOM for you, sometimes you absolutely need access to it. This is the case when you have to use a jQuery plugin or a data visualization library.

There are 2 ways to access the DOM directly. The first way is via ReactDOM.findDOMNode(this). This function takes a mounted component and returns the native browser DOM element. The key word here is "mounted". Therefore, in order to use it you want to attempt to access the DOM once the component has been mounted. This is where lifecycle methods come into play. We will use componentDidMount() and componentDidUpdate() to manually use a jQuery plugin.

The second way is through refs. You can add a "ref" attribute to any JSX element and be able to access it elsewhere in your component with this.refs.theNameOfMyRef.
